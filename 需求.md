# 需求.md

## 核心目标

为 Claude Code 提供 Windows 桌面通知与回焦能力，保证用户在终端外也能及时感知关键状态（任务完成、权限请求、需要输入）。

## 平台约束

1. 仅支持 Windows。
2. Claude Code 保持原生终端运行（不包裹 wrapper）。
3. 事件来源使用 Claude 官方 hooks。

## 第二阶段需求（产品化）

1. 安装与升级可交付（安装包 + CLI）。
2. notifier 无需手动启动，按需自动拉起。
3. `settings.json` 写入必须安全：备份、校验、回滚、可恢复。
4. 卸载可恢复安装前配置。

## 当前进展（2026-04-10）

已完成：

1. Hook 链路升级为 `hook -> bootstrap -> notifier`。
2. notifier 按需启动、单实例约束。
3. CLI：`install/uninstall/doctor/restore/list-backups`。
4. 配置安全机制：
   - 写前备份
   - baseline 记录
   - 写后校验
   - 失败回滚
   - 可按备份恢复
5. 安装器：Inno Setup 打包流程可用。
6. 卸载流程：默认恢复并支持 `--purge`。
7. 自动化测试通过：`35 passed`。
