1. notifier 自动启动

## 2. 当前完成情况（2026-04-10）

1. 架构升级：`hook -> bootstrap -> notifier`。
2. notifier 单实例保护。
3. CLI 命令：

   - `cccue doctor`
   - `cccue restore`

4. 配置安全机制：
   - 写前备份

6. Inno Setup 打包链路可用，已成功产出：
