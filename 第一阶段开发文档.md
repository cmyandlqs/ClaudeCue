# 第一阶段开发文档.md

## 阶段目标

第一阶段目标是建立最小可用链路（MVP）：

`hooks -> notifier HTTP -> 桌面通知`

## 第一阶段已完成项

1. Hook 事件映射与转发。
2. notifier HTTP 服务与基础通知展示。
3. 托盘、声音、回焦基础能力。
4. install.bat 基础配置写入。

## 第一阶段遗留问题（已在第二阶段处理）

1. notifier 需要手工启动。
2. 安装与配置缺少标准化运维入口。
3. `settings.json` 的可恢复性不足。

## 与第二阶段衔接

第二阶段已接管并增强：

1. 新增 bootstrap 按需拉起 notifier。
2. 新增 CLI 与状态管理层。
3. 新增备份/恢复/回滚流程。
4. 新增 Inno Setup 安装包交付路径。
